=========
1) About:
=========

Creates a todo list app for iOS, Android and Web. User
can create todos and cross them off as they do them.
ToDos contain a title, description and date to be completed.

User can edit the todos later or delete them.


============
2) Features:
============

POST - Click on the plus sign on the bottom right.
EDIT - Double tap on a todo to change it.
DELETE - Long press on a todo to delete.
CROSS OUT - Click on a todo to mark it as completed.
